## 🌊 Manifesto

**Float Branding®** é um mindset de design contínuo, inspirado na lógica dos dados do tipo `float`: precisos, interpoláveis, e em fluxo constante. 

Atuamos com branding dinâmico, identidades versáteis e estratégias que respeitam a natureza mutante das marcas no século XXI. Não criamos logos estáticos — desenvolvemos **sistemas visuais runtime**, capazes de adaptar-se sem perder essência, escalar sem distorcer, e evoluir sem traumas.

> *"Se sua marca é líquida, o branding precisa ser flutuante."*

---

## 🧬 O Conceito

### A Metáfora do `float`

Em programação, um `float` é um tipo de dado que representa números reais com ponto flutuante. São valores:
- **Imprecisos mas adaptáveis**: aceitam margem de erro em prol da fluidez
- **Interpoláveis**: transitam suavemente entre estados A e B
- **Contínuos**: não há gaps entre um valor e outro

Aplicamos essa lógica computacional ao design estratégico. Nossas identidades não são estáticas (int), são **flutuantes**: respiram, pulsam, adaptam-se aos contextos sem quebrar.

### Interpolação Visual

```
Estado A ──────────────────> Estado B
   │                            │
   └──── Interpolação Float ────┘
        (suavidade, lógica, flow)
```

## ⚡ Filosofia de Design

### 1. Mutabilidade Estruturada
A marca não é um monumento. É um **organismo** que responde a inputs ambientais: mídia, cultura, tecnologia. Criamos sistemas modulares que permitem mutação controlada.

### 2. Precisão Adaptativa
Como um float de 32 bits, buscamos o equilíbrio entre precisão e performance. Não há rigidez absoluta — há **consistência fluida**.

### 3. Transmedia Nativo
Identidades concebidas para existir simultaneamente em múltiplos planos: físico, digital, espacial (AR/VR), e temporal (temporadas, eventos).

### 4. Código como Estética
O front-end não é apenas implementação; é expressão. Utilizamos:
- Canvas API para generatividade
- CSS Houdini para layout mágico  
- WebGL para experiências imersivas

## 🛠 Stack Tecnológica

| Camada | Tecnologia | Função |
|--------|------------|---------|
| **Visual** | `Canvas 2D API` | Sistema de partículas interativas |
| **Estilo** | `CSS3` + `PostCSS` | Glassmorphism, grids fluidos, variáveis |
| **Motion** | `GSAP` / `Vanilla JS` | Interpolações suaves, observadores de scroll |
| **Tipografia** | `Space Grotesk` + `Inter` | Hierarquia técnica e legibilidade |
| **Lógica** | `Intersection Observer` | Reveals assíncronos, lazy loading estético |
